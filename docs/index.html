About SyBA
SyBA is a tool and database for standardizing gene names in bacterial genomes.
The authors of this tool are Luan Rabelo, Davidson Sodr√©, Luiza Helena Meller da Silva, Grazielle Gomes, Iracilda Sampaio, and Marcelo Vallinoto
SyBA's database is based on 1,215,109 functional bacterial genomes, including the genres Campylobacter, Clostridium, Escherichia, Listeria, Salmonella, Shigella, Staphylococcus, and Vibrio
This tool is available on GitHub, under the MIT license, and offers a database of synonyms, a python class to standardize and search for sequences or manuscripts, using all possible variations of the same gene name, and a web form for isolated searches in public repositories, such as GenBank and PubMedCentral.
